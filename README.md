
# Projecto NLP

Proyecto de chatbot


## Requerimientos

- python >= 3.9
- node >= 20.11.1
- npm >= 10.2.4
- nltk == 3.8.1


## Instalar
```
  git clone https://github.com/fearfulScarab42/proyectoNLP.git
```


```
  pip3 install nltk
```


```
  npm install express
```


## Ejecutar 

```
  cd proyectoNLP
```


```
  node server.js
```

## Acceder

```http
  GET http://127.0.0.1:3000
```
